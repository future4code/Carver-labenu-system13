## LabenuSystem:

API que representa o sistema de ensino.

Possui 3 entidades importantes:

1. Estudantes 

2. Docente

3. Turma

As funcionalidades são:

→ Criar estudante;

→ Criar docente;

→ Criar turma;

→ Adicionar estudante na turma;

→ Adicionar docente na turma;
